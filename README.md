# OS-Simulator for CS255

## Requirements
Django 2.x

## Installation guide
Clone the repo <br>
``` git clone https://github.com/viveshCodes/OS-Simulator.git ```
## Running server 
In the root directory of the repo, hit the following command <br>
``` python manage.py runserver ```
